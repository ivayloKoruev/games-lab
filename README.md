# Games-Lab 🍋🍍🍊

A personal playground focused on building slot games and match-3 games with clean architecture and async logic.
🚧 In progress


## Tech-Stack 🧑‍💻⚙️

- JavaScript (ES6+)
- HTML/CSS
- Pixi.js
- TypeScript
- React.js


## About This Repository

This repository contains small to mid-size game projects built for learning and portfolio purposes.

The main goals are:
- Practice real-world game logic
- Apply async JavaScript patterns
- Separate game logic from rendering
- Build reusable systems for slot and match-3 games

## Running the Project Locally

1. Clone the repository:
   git clone https://github.com/ivayloKoruev/games-lab.git

2. Open the project folder:
   cd games-lab

3. Start a local server (example):
   npm install
   npm run dev
